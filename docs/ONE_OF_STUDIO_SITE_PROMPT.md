# Промпт для разработки сайта ONE OF STUDIO

Сделай статичный одностраничный сайт для танцевальной студии ONE OF STUDIO, г. Хабаровск.

Следуй правилам из `CLAUDE.md`. Перед кодом прочти раздел «Материалы проекта» ниже и все референсы.

## Цель

Сайт должен знакомить посетителя со студией и вести к записи на занятия. Посетитель должен быстро понять:

- что это за студия и чем она отличается;
- какие направления танцев есть и для кого они;
- как и на каких условиях записаться;
- как связаться или найти студию в соцсетях.

Все тексты на сайте — на русском языке. География: Хабаровск.

## Технический формат

- HTML/CSS/JS, один файл `index.html` + отдельный `style.css`.
- Сайт работает без backend.
- Форм нет — контакты только внешними ссылками (Telegram, WhatsApp, телефон).
- Аналитику не подключать.
- Cookie banner не показывать.
- Добавь базовое SEO: `title`, `description`, Open Graph, семантические секции, иерархию H1/H2/H3, alt-тексты, JSON-LD `LocalBusiness`.
- Изображения: `loading="lazy"` для не-hero, явные `width`/`height`, `preload` для hero.
- После реализации запусти локальный сервер, проверь desktop и mobile скриншотами.
- После каждой готовой секции — git commit и push в `main`.

## Визуальное направление

Тёмное, энергичное, editorial. Студия позиционируется как серьёзное место для людей, которые хотят танцевать на высоком уровне. Минимум украшений — максимум характера. Типографика работает как визуальный элемент: огромные заголовки, резкий контраст весов.

Атмосфера: между концертным плакатом и luxury-брендом. Не агрессивно, но уверенно.

Палитра (дизайн-токены):

| Токен | Hex | Роль |
|---|---|---|
| `--bg` | `#0d0d0d` | Основной фон |
| `--surface` | `#261F1F` | Карточки, тёмные секции |
| `--surface-alt` | `#1a1212` | Альтернативный фон секций |
| `--accent` | `#591515` | Основной акцент (тёмный красный) |
| `--accent-bright` | `#400E0E` | Глубокий бордовый |
| `--accent-hover` | `#7a1c1c` | Hover-состояние акцента |
| `--text-primary` | `#F2F2F2` | Основной текст |
| `--text-secondary` | `#A6A4A4` | Второстепенный текст |
| `--border` | `rgba(242,242,242,0.1)` | Разделители |
| `--white` | `#ffffff` | Чистый белый для заголовков |

Типографика:

- **Заголовки (H1, крупные display):** Ethnocentric или Eurostile Extended — bold/heavy, uppercase, condensed. Если недоступны через Google Fonts, использовать `Barlow Condensed` 800 или `Oswald` 700 как замену.
- **Подзаголовки / nav:** Montserrat или `Inter` — uppercase, letter-spacing 0.15–0.25em, weight 500–600.
- **Тело текста:** Montserrat Regular или Inter Regular, 15–16px, line-height 1.6.

## Логотип

Воссоздай логотип в коде (SVG или чистый CSS-текст), не используй картинку:

```
ONE OF          ← огромный, bold, condensed, белый
STUDIO          ← мелкий, uppercase, letter-spacing 0.4em, белый
```

Пропорции как в референсе: «ONE OF» занимает 80–90% ширины логотипа, «STUDIO» под ним центрировано с большой разрядкой.

## Критерии готовности

- Сайт выглядит как premium urban танцевальная студия — не шаблон, не типовой лендинг.
- Первый экран даёт бренд, слоган и навигацию — сразу понятно куда ты попал.
- Цветовая схема строго из палитры: чёрный/тёмный фон, белый текст, красные акценты точечно.
- Типографика выразительная: заголовки занимают пространство, создают характер.
- Нет форм и сбора данных — только ссылки в контактах.
- Нет горизонтального overflow, наложений текста, мелких кнопок, нечитаемого контраста.
- Проверены desktop (1440px) и mobile (430px) через локальный сервер и скриншоты.
- После каждой готовой секции — commit + push.
