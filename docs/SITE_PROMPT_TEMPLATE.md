- [Специфический критерий проекта 1].
- [Специфический критерий проекта 2].
- Нет форм и сбора данных [или: данные собираются только через [сервис]].
- Нет горизонтального overflow, наложений текста, мелких кнопок, нечитаемого контраста.
- Проверены desktop и mobile через локальный сервер и скриншоты.
- Lighthouse-ориентиры: Performance, Accessibility, Best Practices, SEO без очевидных провалов.
